# SaraaTEK - Repair Management System

## Quick Start

### Desktop (Windows)
1. Double-click `StartSaraaTEK.bat`
2. Login: `admin` / `admin123`

### Mobile (Android APK)
1. Right-click `BuildAPK.bat` → Run as administrator
2. Wait for build to complete
3. APK will be in `installer/SaraaTEK.apk`

### Mobile Web App (No build needed)
1. Double-click `WebApp.bat`
2. Open Chrome: http://localhost:4173
3. Chrome menu → "Install app"
4. Works offline like native app!
